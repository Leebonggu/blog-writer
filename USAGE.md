# 네이버 맛집 블로그 글 생성기 — 사용 가이드

## 시작하기

```bash
cd /Users/leebonggu/Desktop/playground/naver-blog-writer
npm run dev
```

브라우저에서 `http://localhost:3000` 접속.

## 환경변수

`.env.local` 파일에 API 키 설정:

```
ANTHROPIC_API_KEY=sk-ant-...   # Claude Sonnet 사용 시 (추천)
OPENAI_API_KEY=sk-...          # GPT-4o 사용 시
```

## 사용 흐름

### 1. 카테고리 선택

상단 탭에서 **맛집 / 배달 / 소품샵** 중 선택.

### 2. 기본 정보 입력

| 항목 | 필수 | 설명 |
|------|------|------|
| 상호명 | O | 가게/브랜드 이름 |
| 네이버 지도 URL | O | `naver.me/...` 단축 URL 또는 `map.naver.com/...` 풀 URL 모두 가능 |
| 정보 가져오기 | - | 버튼 클릭 시 주소, 영업시간, 메뉴, 카테고리, 전화번호 자동 수집 |
| 내돈내산/협찬 | O | 협찬 선택 시 업체명 입력란 추가 |

> 크롤링 실패 시 수동 입력 폼이 자동으로 나타남.

### 3. 이미지 업로드

- 드래그 & 드롭 또는 클릭하여 업로드
- 최대 20장
- 업로드 순서 = 글에서 배치 순서 (IMAGE_1, IMAGE_2, ...)
- 썸네일 hover 시 X 버튼으로 개별 삭제 가능

### 4. 글 옵션

| 옵션 | 설명 |
|------|------|
| 톤 선택 | 친근체 / 정보전달형 / 감성체 / 레퍼런스 글 기반 |
| 재방문 의사 | 꼭 재방문 / 기회되면 / 재방문 없음 — 총평에 반영됨 |
| 필수 포함 문구 | 협찬사 요청 키워드 등. 글에 자연스럽게 녹아듦 |
| 이미지 비전 분석 | ON: 이미지를 AI가 분석해서 묘사 생성 (비용 약간 증가) |
| LLM 선택 | Claude Sonnet (추천) / GPT-4o |

### 5. 글 생성

"글 생성하기" 클릭 → 15~40초 대기 (옵션에 따라 다름).

### 6. 결과 활용

생성 완료 후:

1. **제목 복사** → 네이버 블로그 제목란에 붙여넣기
2. **리치텍스트 복사** → 네이버 블로그 본문에 Ctrl+V (서식 유지됨, h2 큰 글씨 포함)
3. **이미지 배치 가이드** 참고하여 이미지 직접 업로드
4. 마음에 안 들면 "다시 생성하기"

> **텍스트 복사**는 서식 없는 순수 텍스트용.

## 이미지 배치 방법

생성된 글에 `[이미지 1: 외관 사진]` 같은 가이드가 포함됩니다.
네이버 블로그 에디터에서:

1. 리치텍스트 붙여넣기
2. 가이드 텍스트 위치에 커서 놓기
3. 해당 이미지 업로드
4. 가이드 텍스트 삭제

## 생성 글 구조 (맛집 기준)

```
[제목] — 별도 복사

본 포스팅은 직접 방문하여 작성한 솔직한 후기입니다.

인트로 (인사 + 소개 + 훅)
[이미지 1: 대표 사진]

<h2>📍 가게 정보</h2>
📍 위치 / ⏰ 영업시간 / ☎️ 전화번호 / 🚘 주차
[이미지 2: 외관]

<h2>매장 분위기</h2>
인테리어, 좌석, 첫인상
[이미지 3~4]

<h2>메뉴 리뷰</h2>
메뉴별 (사진 → 설명) 반복
[이미지 5~8]

<h2>총평</h2>
만족도, 재방문 의사, 추천 대상
[이미지 9]

── 구분선 ──
주소, 네이버 지도 링크
```

## 비용 참고

| 옵션 | 1회 비용 |
|------|---------|
| 비전 OFF | ~$0.04 |
| 비전 ON (이미지 10장) | ~$0.09 (Claude) / ~$0.13 (GPT-4o) |
| 레퍼런스 글 분석 추가 | +~$0.02 |

## 새 카테고리 추가하기

1. `features/blog/templates/` 에 `{category}.ts` 파일 추가
2. `features/blog/templates/index.ts`의 `CATEGORIES` 배열에 등록
3. `components/blog-form/BlogForm.tsx`의 카테고리 버튼 배열에 추가

## 기술 스택

- Next.js (App Router) + TypeScript + Tailwind CSS
- Claude SDK / OpenAI SDK
- 네이버 지도 크롤링 (m.place.naver.com Apollo State 파싱)
- DB 없음 (모든 상태 클라이언트 메모리)
